    Installation

    It's easy, just link mongo_hacker.js to .mongorc.js in your home directory.

    The manual way:

    $ git clone git@github.com:TylerBrock/mongo-hacker.git
$ cd mongo-hacker
$ ln -sf mongo_hacker.js ~/.mongorc.js

    The automatic way:

    $ git clone git@github.com:TylerBrock/mongo-hacker.git
$ cd mongo-hacker
$ make

    Features

    Colorized Shell

    Aggregation Framework Helpers

    Group and Count
    gcount(group_field, filter)
    Group and Sum
    gsum(group_field, sum_field, filter)
    Group and Sum
    gavg(group_field, avg_field, filter)

    Sensible Defaults

    • A prompt that shows current database, version, and process name
    • Verbose shell enabled (shows query timing information)
    • Auto multi update (works just like update in SQL)
    • Two spaces indent (why waste the space?)
    • Colon next to keys (looks nicer right?)

    Index Paranoia

    Print index usage information on every query

    Colorize and Highlight Query Time

    Highlights slow running queries
